# Voice-Based UPI System for Visually Impaired 🎙️

This project is designed to help visually impaired users perform digital payments using voice commands.

## 💡 Problem
Visually impaired users face difficulty using mobile-based UPI apps due to heavy visual interfaces.

## 🚀 Solution
A voice-driven interface that allows users to send money using simple speech commands.

Example:
"Send 500 rupees to Ravi"

## 🔧 Features
- Speech-to-text input
- Command recognition
- Voice confirmation
- Simple transaction simulation

## 🛠 Tech Stack
- Python
- SpeechRecognition
- Text-to-Speech (pyttsx3)

## 🚧 Status
Prototype version (under development)

## 🎯 Goal
To build an accessible and easy-to-use payment interface for visually impaired users.
